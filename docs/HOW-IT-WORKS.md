# DESTINY 是怎麼算出你們的配對的？

這份文件用白話說明 DESTINY 的配對邏輯，讓你了解分數背後代表什麼意思，
以及系統是怎麼「認識」你這個人的。

---

## 我們不看臉，我們看「源代碼」

大多數交友 App 靠照片和自我介紹配對——這些都是你**選擇展示**的樣子。
DESTINY 從另一個角度切入：出生時刻烙印在你身上的宇宙格局、你面對衝突
和親密關係的本能模式，以及你的心理依戀風格。

這些不是你能「表演」的，是你天生帶來的。

---

## 六個維度

### 1. 占星盤相（星座行星位置）

你出生的那一刻，每顆行星正落在某個星座。兩個人的行星格局相遇，
會產生不同的「化學反應」。

DESTINY 把行星分成兩類來計算：

**和諧型行星**（月亮、水星、木星、土星、金星）
> 問的是：你們在一起舒不舒服？有沒有共同的頻率？

當這些行星的星座彼此「順暢」（例如相差 120°，占星稱為三分相），
代表你們的生活節奏、溝通方式、價值觀天然合拍，相處不費力。

**張力型行星**（火星、冥王星、凱龍星）
> 問的是：你們之間有沒有火花？有沒有「不得不」靠近的磁力？

這裡反過來——當這些行星「衝突」（相差 90°，占星稱為四分相），
反而代表強烈的性張力和征服慾。太順了，反而少了激情。

特別重要的是**跨人相位**：不是你自己的火星和金星，
而是**你的火星對上對方的金星**。這個「一個人追、另一個人被吸引」的動態，
是系統判斷原始吸引力最核心的計算。

相位越精準，力量越強——角度差在 1° 以內是最強的訊號；
差距越大，效力越低。

這就是為什麼有些人在一起「很舒服但沒感覺」，
有些人「明明處處磨合，卻死命離不開對方」——背後都有結構。

### 2. 業力觸發器（Shadow & Synastry Triggers）

除了基本行星相位，DESTINY 還會偵測一系列「命運級別」的特殊觸發點。
這些觸發器使用**黃道精確度數**計算，比一般星座比對更準確，
也是讓系統給出「⚡ 高壓警告」的主要來源。

#### 傷口與禁忌類

**凱龍傷口觸發（Chiron Wound）**

凱龍星代表每個人最深的心理創傷點。當某人的太陽、月亮、金星或火星，
精確落在對方的凱龍星附近（5° 以內，合相或對沖），
代表這個人的存在本身就能喚醒對方的傷——這是最典型的「虐心吸引」。
火星 / 冥王星觸發凱龍時，還會加劇慾望面的磁力。

**冥王月亮深層傷口（Pluto-Moon Wound）**

當某人的冥王星精確合上對方的月亮（5° 以內），
代表一種深入骨髓的情感穿透——對方能觸及你最原始的情緒底層。
這個組合同時推高靈魂深度和慾望強度，但也削弱伴侶穩定感。

**莉莉絲禁忌觸發（Lilith）**

暗月莉莉絲代表本能的、被壓抑的原始慾望。
當某人的金星或火星精確合上對方的莉莉絲（3° 以內），
就是那種「說不清楚但忍不住」的禁忌吸引——這是 ⚡ 高壓警告的最強觸發之一。

#### 命運與宿命類

**宿命點觸發（Vertex）**

宿命點（Vertex）是命盤裡的「命定門戶」。當某人的太陽、月亮或金星精確落在
對方的宿命點附近（3° 以內），算命學上稱為「命中注定的相遇」。
這不製造痛苦，但製造「莫名熟悉感」和強烈的宿命感。

**南北交點業力觸發（Lunar Nodes）**

南交點代表前世的業力積累；北交點代表今生的靈魂進化方向。
當某人的行星觸碰到對方的南交點（3° 以內），代表你們可能帶著
「前世未完成的課題」重新相遇——宿命感極強，但也容易陷入輪迴式的關係模式。
觸碰北交點則代表這個人能催化你的靈魂成長。

**下降點（Descendant）婚姻宮觸發**

下降點就是你命盤的「伴侶宮」——它描繪你靈魂深處理想伴侶的形象。
當某人的太陽、月亮或金星精確合上你的下降點（5° 以內），
代表他就是你潛意識中一直在尋找的那個人。
這是少數同時提升伴侶穩定感和靈魂連結的觸發器。

**12 宮陰影疊圖（12th House Overlay）**

當某人的太陽、火星、月亮或金星落入對方的第 12 宮，
代表一種潛意識層面的連結——你可能說不出為什麼，
但總覺得對方像是你夢境裡的人。雙向 12 宮觸發更是極為罕見的靈魂共振。

#### 土星控制類（權力結構觸發器）

土星代表紀律、規則和壓制。當它觸碰到對方的個人行星，
會形成一種不對等的「管教者 vs 被管教者」動態：

**土星-月亮（情感壓制）**：一方的土星合/刑/沖對方的月亮，
對方的情緒表達會被無形地壓制——學會隱藏脆弱，表面平靜內心翻湧。

**土星-金星（愛慕壓制）**：一方的土星壓制對方的金星，
對方會覺得自己的愛不夠好、不值得被愛，慾望感也隨之冷卻。

**土星-太陽（權威壓制）**：一方的土星壓制對方的太陽，
對方在你面前變得謹慎、壓抑核心自我——你的認可是他最渴望卻最難得到的東西。

**土星-火星（行動壓制）**：一方的土星壓制對方的火星，
對方的衝動與慾望被無形的韁繩拉住——想衝、想要，卻不自覺地收斂。

#### 上升磁場類

**火星/金星-上升點（ASC Magnetism）**

當某人的火星或金星精確合上對方的上升星座（5° 以內），
代表你的吸引力恰好契合對方的「第一印象濾鏡」——
火星觸發上升是「一見鐘情式的衝擊感」，金星觸發上升是「賞心悅目的和諧感」。

> 有精確出生時間（Tier 1）才能計算 Vertex、Lilith、Descendant、
> 12 宮疊圖和上升磁場觸發器。

### 3. 八字日主五行

你的生日決定了你的「日主五行」——這是你生命力的核心屬性：
木、火、土、金、水。

五行之間有三種關係：

- **相生**（例如木生火）：一方滋養另一方，像水澆灌了土地。
  這種組合最容易建立深度的伴侶感和長期承諾。

- **相剋**（例如金剋木）：一方制約另一方，像刀刃碰上木頭。
  聽起來不好，但這種張力往往燃燒出最強烈的化學反應和激情。

- **比和**（相同五行）：能量共鳴，靈魂層面的熟悉感，
  像在對方身上看見了另一個自己。

更進一步，系統會分析你的八字命局中哪個五行最缺乏（「用神」），
如果對方的日主五行恰好能補足你的用神，代表他的存在本身就能
為你帶來平衡——這是一種超越表面相性的深層互補。

### 4. 關係權力動態（RPV）

透過三個簡單問題，系統了解你的關係本能：

- **衝突時，你更傾向？** 沉默冷戰，還是直接把話說開？
- **在關係裡，你更想？** 主導方向，還是配合對方？
- **你的能量偏好？** 待在家窩著，還是出去探索世界？

這不是在評判你好不好——每種風格都有它的魅力。
系統在乎的是：兩個人的模式**互補**嗎？

一個喜歡主導的人，配上一個享受被帶著走的人，
往往能建立清晰、省力的關係結構。
兩個都想主導的人，則需要更多意識去處理張力。

除了問卷本身，星盤中的某些觸發器也會影響權力動態的判定：
- **冥王星權力壓制**：當某人的冥王星 hard-aspect 對方的太陽、月亮或金星，
  代表這個人在關係裡天然帶有「控制場」——不需要刻意，
  存在本身就讓對方感到被壓制或被吸引。
- **凱龍傷口**：當火星或冥王星觸碰對方的凱龍星，
  也會放大一方的權力框架——因為傷口被喚醒的人，
  往往不自覺地將權力交給觸發者。

### 5. 依戀心理學

依戀風格來自發展心理學的研究——它描述你在親密關係中最深層的本能：

- **安全型（Secure）**：你能自在地靠近對方，也不怕對方需要空間。
- **焦慮型（Anxious）**：你渴望親近，害怕被遺棄，情感的波動很敏感。
- **迴避型（Avoidant）**：你重視自主，對太快靠近的關係本能地後退。

最穩定的配對是兩個安全型（系統稱為「Safe Haven」，同時提升靈魂和伴侶分數）。
而焦慮型 + 迴避型的組合（「Anxious-Avoidant Trap」），
往往製造出那種「痛苦但上癮」的動態——
不是因為不好，而是因為彼此的依附模式剛好互相觸發，
慾望飆升但伴侶穩定感被大幅削弱。

你的依戀風格不是一成不變的——星盤中的某些行星可以改變你的基本模式：
- **天王星**刑沖月亮：讓安全型也帶上迴避傾向（獨立需求增強）
- **土星**刑沖月亮：壓抑情緒，讓焦慮型變得更焦慮或被迫表現迴避
- **木星**合月亮：擴展情感容量，讓迴避型也能展現安全型的特質

系統會根據這些行星動態，在你填入的基礎風格上做微調，
讓依戀匹配更貼近你真實的關係表現。

### 6. 紫微斗數（ZiWei DouShu）

這是三套算法裡最古老的那一套，也是最能看見「命運羈絆」的系統。

> **Note:** 紫微斗數只對有精確出生時間（Tier 1）的用戶啟動；若你
> 沒有填入精確時間，前五個維度依然完整計算。

紫微斗數把你的命盤分成 12 個宮位，三個機制影響配對結果：

**命宮主星（你天生的關係人設）**

命宮主星決定你在感情裡的本能傾向：

- **殺破狼**（七殺、破軍、貪狼）：叛逆高張力，需要持續的刺激和衝突感，
  太安穩反而讓你感到窒息。
- **紫府武相**（紫微、天府、武曲）：主導型，視穩定為根本，
  你在關係裡天生是「設定規則的那個人」。
- **機月同梁**（天機、太陰、天同）：細膩敏感，精神交流比肉體吸引
  更重要，你在找能讀懂弦外之音的靈魂知己。

**飛星四化（你的能量如何影響對方）**

每個人的出生年天干，決定了四顆「化星」落在哪顆星上。
如果你的化祿（代表滋養）落入對方的命宮或夫妻宮，
那一段關係天然充滿「給予與被滋養」的流動——伴侶感大幅提升。
如果是化忌（代表業力執念）雙向觸發，那就是傳說中的「業力虐戀」：
靈魂層面強烈連結，但也更容易在關係裡反覆糾纏。

**夫妻宮煞星（壓力下的本能反應）**

夫妻宮的煞星揭示你在關係受壓時的防禦機制：
- 擎羊 / 火星 → 先發制人（衝突時第一個開口）
- 陀羅 / 鈴星 → 沉默反芻（把話吞回去慢慢消化）
- 天空 / 地劫 → 突然抽離（情感波動時本能地消失）

## 兩個核心分數

### VibeScore（肉體吸引力）

這個分數代表：你們之間有多少**慾望和磁力**。

計算的核心訊號是**跨人交叉相位**：A 的火星與 B 的金星之間的角度，
以及 B 的火星與 A 的金星之間的角度。這個「追求 vs 被追求」的動態，
是系統判斷原始吸引力最強的訊號。

其次是同行星相位（雙方的金星對金星、火星對火星），
反映美感頻率與慾望節奏是否同步。

角度越精準，訊號越強——0° 的完美合相得滿分；
相差超過 8°，效力就會大幅衰減。

如果你有填精確出生時間，還會加入：
- 第 8 宮的影響（占星中，第 8 宮管的正是親密、慾望和禁忌）
- 火星 / 金星對上對方上升星座的能量（你的吸引力如何被「看見」）
- 莉莉絲禁忌觸發、凱龍傷口觸發、冥王月亮深層傷口

八字五行也會影響這個分數——如果你們的日主五行是**相剋**關係，
VibeScore 會乘以 1.25 的加成，因為相剋本身就是一種磁力。

### ChemistryScore（靈魂連結）

這個分數代表：你們之間有多少**情感深度和長期潛力**。

計算時看的是月亮（情緒共鳴）、水星（溝通頻率）、土星（承諾感）、
婚神星 Juno（伴侶連結），以及你們的依戀風格是否相容。

如果有精確時間，還會加入宿命點（Vertex）觸發、南北交點業力觸發、
下降點（Descendant）婚姻宮觸發，讓「命定感」更立體。

八字五行如果是**相生**關係，ChemistryScore 會乘以 1.20 的加成，
因為相生代表一方滋養另一方的靈魂節奏。

## 四條關係軌道

系統會同時計算你們在四種關係類型上的得分：

| 軌道 | 代表什麼 |
|---|---|
| **朋友軌** | 你們的智識連結、共同頻率、相處的自在感 |
| **激情軌** | 你們之間的慾望、磁力、那種「說不清楚的吸引」 |
| **伴侶軌** | 你們的穩定感、共同建立生活的潛力 |
| **靈魂軌** | 你們之間深層的命運感、彼此的成長觸發 |

哪條軌道得分最高，就是你們這段關係的「主旋律」。
高激情軌的兩個人，在一起會燃燒；
高伴侶軌的兩個人，在一起會安心。

伴侶軌還有一個「情緒容量」守門機制——如果雙方的情緒穩定性都偏低，
伴侶軌會被適度下修，反映「兩個情緒海綿放在一起會彼此消耗」的現實。

如果你們都有精確出生時間，紫微斗數會進一步調整這四條軌道的比重——
命宮主星和飛星四化可以大幅放大某條軌道，讓「主旋律」更清晰。

## 你們在哪個象限？

用 VibeScore 和 ChemistryScore 畫一個座標軸：

```
高 ChemistryScore
       │
穩定伴侶 │ 靈魂伴侶
（Partner）│（Soulmate）
       │
───────┼────────── 高 VibeScore
       │
命定過客 │ 命定雙生
（Colleague）│（Lover）
       │
```

沒有哪個象限「最好」——不同的關係需求，適合不同的象限。
有人想要燃燒，有人想要安穩，有人兩者都要。

系統還會計算一個綜合的**緣分指數（Harmony Score）**：

```
Harmony Score = VibeScore × 40% + ChemistryScore × 60%
```

靈魂連結的權重比肉體吸引高——因為長期關係裡，靈魂共鳴比慾望更持久。

系統還會給出一個**辣度等級（Spiciness Level）**：

| 等級 | 描述 |
|---|---|
| **STABLE** | 平穩的日常，細水長流 |
| **MEDIUM** | 有明顯的吸引或靈魂共鳴 |
| **HIGH_VOLTAGE ⚡** | 莉莉絲禁忌、12宮陰影疊加、冥王月亮、業力南交點、焦慮-迴避陷阱等強烈觸發——上癮感極強，修羅場體質 |
| **SOULMATE** | 靈魂與穩定感同時爆表，稀有的宿命感連結 |

## 業力軸線：你這一世的靈魂課題

北交點和南交點構成你命盤的「業力軸線」，揭示你的靈魂進化方向：

- **南交點**：你前世帶來的天賦與習慣，也是你容易重蹈覆轍的地方。
- **北交點**：你這一世需要發展的方向，雖然陌生，卻是靈魂的成長線。

例如，北交點在處女座的人，這一世的課題是把靈感落地為行動；
而他們容易逃進北交點對面的雙魚座——用靈性夢境逃避現實責任。

系統還會依據精確出生時間，計算北交點落在哪個**宮位**，
讓這個靈魂進化的方向更加個人化。

## 出生時間重要嗎？

重要，但你不填也沒關係。

- **有精確出生時間（Tier 1）**：系統能計算上升星座、下降點（Descendant）、
  第 4 宮、第 8 宮，以及宿命點（Vertex）和暗月莉莉絲（Lilith）。
  同時啟動完整的紫微斗數合盤分析（飛星四化、命宮主星、煞星防禦），
  以及業力軸線的宮位精確計算。
  這是分數最完整、最個人化的模式。

- **只知道大概時段或不知道（Tier 2/3）**：系統會聰明地降級，
  專注在移動緩慢的星體——婚神星 Juno 和凱龍星 Chiron 一個星座
  待好幾個月甚至好幾年，不需要出生時間就能精準計算。
  西洋占星、業力軸線星座方向和八字五行的配對依然完整運算，

## 關於隱私與演算法限制

DESTINY 的演算法是一個**輔助工具**，不是命運裁判。

- 分數高不保證關係一定成功；分數低不代表兩人沒有可能。
- 真實的關係受太多變數影響：成長、選擇、時機、雙方的意願。
- 你的出生資料僅用於計算，不會用於其他商業用途。
